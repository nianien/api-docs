Zero Markets 支持多种交易平台（以实际情况为准）：

## MetaTrader 4 / 5

- **MT4**: 经典交易平台，适合所有交易者
- **MT5**: 更先进的平台，支持更多工具和指标

### Features
- 高级图表分析
- 自动化交易（EA）
- 移动端应用
- 多账户管理

## WebTrader / Mobile App

- 无需下载，浏览器直接使用
- 响应式设计，适配所有设备
- 实时报价和图表

## API Trading

- **REST API**: 标准 HTTP 接口
- **FIX Protocol**: 专业交易接口
- 查看 [API Reference](/api/trading) 获取详细信息

每种平台的下载地址、登陆方式与注意事项可以在官网或本开发者门户中找到。
