Sandbox / Demo Environment

Zero Markets 提供 Sandbox / Demo 环境，用于无风险地测试你的集成与交易策略。

1. 环境对比​

环境Base URL说明
Sandboxhttps://api-sandbox.zeromarkets.com模拟环境，不涉及真实资金
Livehttps://api.zeromarkets.com实盘环境，连接真实账户

除非特别声明，API 的接口路径在两个环境中保持一致，只是域名不同。

2. Sandbox 账户与资金​

  • 你可以通过网页后台创建 Demo Account
  • Demo 账户会自动分配一笔虚拟资金（例如 10,000 USD）
  • 可随时重置账户或新增模拟账户，用于多策略测试

注意：Sandbox 环境中的价格与成交逻辑可能简化，仅用于功能验证，不代表真实市场行为。

3. 可用产品与市场数据​

在 Sandbox 中：

  • 可用产品列表可能少于 Live（例如仅提供主流货币对与少数指数）
  • 行情价格可来自：
    • 延时的市场数据
    • 仿真数据流
  • 部分风控规则可能被放宽，仅用于验证接口

建议：

  • 使用 GET /v1/instruments 获取当前环境支持的品种列表
  • 不要在 Sandbox 中依赖价格进行正式回测

4. 限流与配额​

Sandbox 环境通常会有独立的限流策略，例如：

  • 每秒 X 次请求
  • 每分钟 Y 次下单

具体限制以 API 返回的 429 Too Many Requests 与相关响应 Header 为准。
如果需要更高配额用于压力测试，请联系技术支持。

5. 差异与迁移建议​

从 Sandbox 迁移到 Live 时，请注意：

  1. Base URL 替换

    • 从 api-sandbox 换到 api

  2. 凭证隔离

    • Live 使用单独的 API Key / Secret，不复用 Sandbox 凭证

  3. 风险参数

    • 实盘账户的杠杆、保证金要求可能与 Demo 不同

  4. 风控与监控

    • 在正式接入前，请在 Live 中小额测试，并建立监控和告警

强烈建议：先在 Sandbox 完成所有接口联调与自动化测试，再以小规模实盘验证，最后再扩大规模。
