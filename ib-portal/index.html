IB Portal Open API (1.0.0)

Download OpenAPI specification:Download

Open API for querying referral code accounts, trading history, and rebate history

Referral

Manage referral code level entities.

Query account list by referral code

Returns trading accounts mapped to the provided referral codes.

Authorizations:
ApiKeyAuth
query Parameters
referralCodes
required
Array of integers <int64> [ items <int64 > ]
Example: referralCodes=10001,10002

Comma-separated referral codes. Maximum 20 per call.

includeSubIB
boolean
Default: false

Include sub-IB accounts when true.

Responses

https://gateway.zeromarketsapi.com/v1/listReferralCodeAccount

Request samples

  • Fetch referral accounts
curl --request GET \
  --url 'https://gateway.zeromarketsapi.com/v1/listReferralCodeAccount?referralCodes=10001,10002&includeSubIB=true' \
  --header 'Authorization: Bearer <token>'

Trading

Query MT4/MT5 trading activities and rebates.

Query trading history by account list

Returns paginated MT4/MT5 trades for the specified accounts.

Authorizations:
ApiKeyAuth
Request Body schema: application/json
required

Provide the account list and filters. When no date range is supplied, the last three months are queried automatically.

accountList
Array of integers <int64> [ items <int64 > ]
startTime
string <date-time>
endTime
string <date-time>
startDate
string
endDate
string
ifOpenQuery
boolean
includeSubIB
boolean
pageNo
integer <int32>
Default: 1
pageSize
integer <int32>
Default: 5000

Responses

https://gateway.zeromarketsapi.com/v1/listTradingHistory
